Que 3 : 
---------
Write a JavaScript program that finds and prints all pairs of numbers from a given array whose sum is equal to a specified target value. 
The program should iterate through the array, compare every pair of numbers, and print each pair that meets the condition.

Example:
Sample input:
	Array: {1, 5, 7, -1, 5}
	Target Sum: 6

Sample output:
	Pair found: (1, 5)
	Pair found: (7, -1)
	Pair found: (1, 5)

		Armstrong numbers are special numbers that are equal to the sum of their digits raised to the power of the number of digits.

		if user enter 3 as a input the program should print all 3 digit Armstrong numbers.

	Sample Input : 
		Enter a Number : 1
	Sample Output : All 1 digits Armstrong numbers are : 1 2 3 4 5 6 7 8 9

		Enter a Number : 3
